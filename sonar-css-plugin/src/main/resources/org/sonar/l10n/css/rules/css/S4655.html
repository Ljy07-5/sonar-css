关键帧（keyframes）声明中的!important 在某些浏览器中完全被忽略，因此不应该在所有浏览器中使用它。

不合规的代码示例

@keyframes kf {
  from { margin-top: 50px; }
  50%  { margin-top: 150px !important; } /* 不合规的；被忽略的 */
  to   { margin-top: 100px; }
}

合规的解决方案

@keyframes kf {
  from { margin-top: 50px; }
  50%  { margin-top: 150px; }
  to   { margin-top: 100px; }
}
