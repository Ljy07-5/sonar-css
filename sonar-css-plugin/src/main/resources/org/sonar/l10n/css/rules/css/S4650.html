calc 是一个CSS3函数，提供了在CSS中进行简单的数学运算（加， 减， 乘， 除）的可能性。 如果运算符周围 没有空格， calc 将没有效果。

更精确地说，在运算符之前，必须有一个空格或一个换行符增加缩进。在运算符之后，必须有一个空格或一个换行符。

不合规的代码示例

#div1 {
    position: absolute;
    width: calc(100%- 100px); /* 不合规的；在%标志后没有空格 */
}

合规的解决方案

#div1 {
    position: absolute;
    width: calc(100% - 100px);
}
