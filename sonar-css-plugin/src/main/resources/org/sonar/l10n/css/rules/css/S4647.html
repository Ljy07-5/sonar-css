一个无效的颜色定义将被默认理解为黑色，这可能会对网站的预期外观和感觉产生意想不到的影响。

当颜色定义(color, background-color) 是无效的时，此规则会引发一个问题。 当颜色定义是由十六进制字符组成时，它被认为是有效的：

- 普通属性（longhand）：6或8个字符（定义不透明度时）

- 简写属性变型（shorthand variant）: 3或4个字符（定义不透明度时）

不合规的代码示例

a {
 color: #3c; /* 不合规的；简写属性应该由3个字符构成 */
}
div {
  background-color: #3cb371a; /* 不合规的；不透明度定义（alpha）应该包含2个字符 */
}

合规的解决方案

a {
 color: #3cc;
}
div {
  background-color: #3cb371ac;
}
