使用CSS3标准化linear-gradient 。在此之前，可以使用不同的非标准语法来定义渐变的方向。 因为这些语 法不是标准的，所以不是所有的浏览器都支持它们，因此它们不该再被使用，以便在支持CSS3的最新版本浏览器中获得预计的渐变效果。

当 linear-gradient 的第一个参数不是有效的 <side-or-corner> 或者 angle 时，这个规则就会产生一个问题。

不合规的代码示例

.foo {
  background: linear-gradient(top, #fff, #000);
}

.bar {
  background: linear-gradient(45, #fff, #000);
}

合规的解决方案

.foo {
  background: linear-gradient(to top, #fff, #000);
}

.bar {
  background: linear-gradient(45deg, #fff, #000);
}
